#Numpy vs Pandas

##Numpy Array vs Pandas Series
Homogeneous, Heterogeneous

Integer based indexing, Labelled (cutomizable)

No automatic alignment, Aligns by index in operations

Missing data- Limited support, Built-in(NaN,None)

Methods- Basic numeric and array ops, many conventional data manipulation methods(mao, value_counts, fillna,etc)

Use case- High performance numeric computing, Real-world data with labels, missing values, or heterogeneous types
